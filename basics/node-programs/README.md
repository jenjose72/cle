# Node.js Programming Examples

This directory contains 10 different Node.js programs designed to teach you the core concepts of the language, from basic syntax to building a web server and a CRUD API.

## How to Run the Programs
Ensure you have [Node.js installed](https://nodejs.org/) on your system.

To run any of these programs, open your terminal in this directory and use:
```bash
node <filename>.js
```

## Program Overview

1. **01_hello_world.js**: Introduction to Node.js output and template literals.
2. **02_variables_loops.js**: Learn about variable types (`const`, `let`), loops, and array iteration.
3. **03_functions_objects.js**: Traditional functions, arrow functions, and how objects work by reference.
4. **04_classes_inheritance.js**: Object-oriented design in Node.js using ES6 classes and inheritance.
5. **05_error_handling.js**: Using `try...catch`, `throw`, and `finally` for robust error management.
6. **06_async_await.js**: Managing non-blocking tasks with Promises and `async/await`.
7. **07_file_io.js**: Asynchronous reading and writing to the local filesystem using `fs/promises`.
8. **08_basic_server.js**: A simple web server using the built-in `http` module.
9. **09_html_rendering.js**: Serving dynamic, server-side rendered HTML using standard JS templates.
10. **10_crud_app.js**: A complete in-memory CRUD API (GET, POST, PUT, DELETE) using standard modules.

### Note on Web Servers
For programs 8, 9, and 10, once you run them, they will start a server. You can visit them in your browser at:
- **08_basic_server.js**: `http://localhost:3000`
- **09_html_rendering.js**: `http://localhost:3001`
- **10_crud_app.js**: `http://localhost:3002`

Happy Coding!
